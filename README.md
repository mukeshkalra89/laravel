# laravel
1. Make a empty folder and unzip the zip file there
2. Update database details in .env file
3. Run the following commands on project root directory
4. composer install
5. nmp install
6. php artisan migrate
7. php artisan db:seed --class=UserSeeder
8. php artisan db:seed --class=CategorySeeder
9. Login with email: admin@gmail.com and password : pass
10. Add products and check product listing with filters(title, category)
